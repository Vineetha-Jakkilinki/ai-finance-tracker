## Setup Instructions

1. Clone the repository

```
git clone https://github.com/Vineetha-Jakkilinki/ai-finance-tracker.git
```

2. Install dependencies

```
npm install
```

3. Create environment variables

Copy the example file and create `.env.local`.

```
cp .env.example .env.local
```

4. Add your API keys from services like Clerk and Groq.
   
5. Live Demo: https://ai-finance-tracker-five-pi.vercel.app

7. Run the project

```
npm run dev
```
